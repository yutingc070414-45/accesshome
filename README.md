# AccessHome

AccessHome is a full-stack smart home management system designed to help users control home devices remotely.

## Features

- User registration and login
- JWT authentication
- Smart light control
- Air conditioner control
- Curtain control
- Real-time UI interaction
- REST API documentation with Swagger

## Tech Stack

### Frontend
- React
- Vite
- CSS

### Backend
- Spring Boot
- Spring Security
- JWT
- JPA / Hibernate

### Database
- PostgreSQL

### Deployment
- Vercel (Frontend)
- Render (Backend)

## System Architecture

Frontend (React)  
↓  
REST API (Spring Boot)  
↓  
PostgreSQL Database
